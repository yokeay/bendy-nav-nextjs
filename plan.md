- [x] 迁移首页全局通知与用户菜单的基础能力。
- [x] 新版首页设置中心落地，覆盖布局、搜索、时间与外观四组核心配置。

- [x] 新版首页进入编辑模式后，已经支持当前分组内标签编辑/删除、分组管理和拖拽排序。

## 目标说明

- [ ] 目标不是“功能迁移到大致可用”，而是“首页 UI 与交互对原项目 1:1 复刻”。
- [ ] 完成标准：同一数据集下，新版首页与旧版首页在布局节奏、图标尺寸、交互路径、持久化结果上保持一致。
